This is the Quickstart example of Webino the Open Source Website System.

Check out project homepage: http://webino.org/

RELEASE INFORMATION
-------------------
Release: Webino 0.1.0alpha
Released on February 10, 2012.

ABOUT
-----
Please see Webino Quickstart:

http://webino.org/how-to/quick-start/

GETTING STARTED
---------------
Webino requires Zend Framework 1.11.11 and PHP 5.3.3 or later. Please see our
reference guide for more detailed requirements:

http://webino.org/documentation/getting-started/

INSTALLATION
------------
Copy sources to your web server public directory and browse to project
public directory.

QUESTIONS AND FEEDBACK
----------------------
Online documentation can be found at http://webino.org/documentation/
Questions that are not addressed in the manual should be directed to the
Webino Google groups:

https://groups.google.com/group/webino-project-forums/

If you find code in this release behaving in an unexpected manner or
contrary to its documented behavior, please create an issue in the Webino
issue tracker at:

https://github.com/webino/quickstart/issues/new/
